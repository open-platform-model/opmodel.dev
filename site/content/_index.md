# Open Platform Model

A declarative platform model for describing applications and their infrastructure requirements.

## Features

- **Type-Safe** - CUE-based definitions with validation at authoring time
- **Open Source** - All definitions, providers, and tooling under Apache 2.0
- **Well-Documented** - Comprehensive reference documentation generated from code
