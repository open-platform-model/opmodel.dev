Complete reference for all OPM definition types, auto-generated from CUE schemas in the catalog.

## Definition Types

### Core Definitions

- **Module** - A complete application definition
- **ModuleRelease** - A deployed instance of a Module
- **Component** - A composable unit within a Module
- **Resource** - Deployable components and services
- **Trait** - Behaviors that modify Resources
- **Blueprint** - Pre-composed patterns
- **Policy** - Cross-component rules and constraints

### Resources

Deployable components like containers, volumes, and configuration.

### Traits

Behaviors like scaling, health checks, routing, and security.

### Blueprints

Pre-composed patterns for common workloads.

### Providers

Provider-specific transformers (e.g., Kubernetes).
